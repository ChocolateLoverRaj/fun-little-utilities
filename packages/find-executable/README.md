# Find Executable

Search for a name that is executable

## Usage

```
Options:
  --version     Show version number                                    [boolean]
  --filter, -f  Display what?[choices: "word", "path", "both"] [default: "path"]
  -h, --help    Show help                                              [boolean]
```
